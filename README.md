# Iroto Web v2.10

这是 Iroto Web v2.10：拍照功能移除后的文案与导入流程整理版。

## 1. 开始页文案不再出现“拍照 / 撮影”

首页主说明改为：

```text
写真を選択し、
傾けて演奏しましょう。
```

首页底部提示改为：

```text
保存済みの写真を選択してください。
```

中文与英文也同步改为只描述“选择已保存照片”。

## 2. 演奏页替换照片直接打开文件选择

底部“写真を変更”按钮现在直接打开系统文件 / 相册选择。  
不再出现“选择照片 / 拍照”的二选一弹窗。

## 3. 清理残留拍照表述

虽然 v2.9 已经移除了拍照功能，但部分多语言字符串里仍保留了拍照相关词。  
v2.10 已将这些残留表述统一改为选择照片相关文案，避免语言切换或未来修改时误出现。

## 4. 版本号更新

资源路径更新为：

```text
app.js?v=2.10.0
styles.css?v=2.10.0
manifest.webmanifest?v=2.10.0
```

## 5. 保持不变

- v1.6 的矩阵相对旋转传感器逻辑保持不变
- 独立 1080p 录制画布保持
- WebM 时长元数据修正保持
- 日语默认和中日英切换保持
