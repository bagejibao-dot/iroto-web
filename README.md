# Iroto Web v2.16

这是 Iroto Web v2.16：横向加速度稳定化 + 兼容性检查显示版本号版。

## 1. 横向加速度补偿调稳

v2.15 加入加速度补偿后，上下平移基本可用，但左右平移仍容易出现“8 字形”轨迹。  
v2.16 对横向加速度做了更稳的处理：

```text
横向加速度更慢平滑
横向补偿强度略降低
X/Y 轴互相串扰时，优先保留主运动轴
暧昧的斜向/手部旋转输入会被压低
```

这样目标是保留“平移有反应”的体验，同时减少左右移动时的绕圈感。

## 2. 兼容性检查中加入版本号

兼容性检查弹窗底部现在会显示当前版本：

```text
v2.16.0
```

字号较小，不影响原本的检查列表。

## 3. 版本号更新

资源路径更新为：

```text
app.js?v=2.16.0
styles.css?v=2.16.0
manifest.webmanifest?v=2.16.0
```

## 4. 保持不变

- v1.6 的矩阵相对旋转传感器逻辑保持
- 自动全屏保持
- 全屏横屏 safe-area 居中保持
- 点击画面显示/隐藏 UI 保持
- 拍照功能继续移除，只保留照片导入
- 独立 1080p 录制画布保持
- WebM 时长元数据修正保持
