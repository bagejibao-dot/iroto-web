# Iroto Web v2.2

这是 Iroto Web v2.2：横屏计时器位置、控制层透明度与录制画面铺满修正版。

## 1. 横屏计时器位置调整

v2.1 中横屏时计时器会和顶部 UI 重叠。  
v2.2 改为：

```text
竖屏：计时器仍在上方中央
横屏：计时器移动到左侧轴线，并放在屏幕垂直中间
```

这样更接近左侧“语言切换 / 替换照片”轴线，也能减少遮挡准星轨迹。

## 2. UI 整体更透明

进一步降低了顶部栏、底部栏、播放/停止按钮、回正按钮和计时器的背景不透明度。  
演奏中临时显示 UI 时，照片、准星和轨迹会更容易看见。

## 3. 照片与录制视频改为铺满画布

之前照片使用 contain 方式绘制，因此保存的视频会保留黑边。  
v2.2 改为 cover 方式：

```text
保持照片等比例
居中裁切
铺满整个画布
录制视频与演奏画面一致
不再留黑边
```

## 4. 保持不变

- v1.6 的矩阵相对旋转传感器逻辑保持不变
- 录制计时器功能保持
- 日语默认和中日英切换保持
- 保存信息多语言和删除二次确认保持
- Service Worker 仍然禁用，便于 Netlify 测试
