# Iroto Web v2.11

这是 Iroto Web v2.11：照片显示修复 + 权限文案统一版。

## 1. 修正“选了照片后只显示 UI、不显示照片”

v2.10 移除拍照功能后，`applyLanguage()` 中仍残留了对已删除按钮的引用，例如：

```text
cameraBtn
sourceAlbumBtn
sourceCameraBtn
```

这些元素已经不存在，会导致脚本初始化时报错，进而影响画布绘制。  
v2.11 已清理这些残留引用，照片导入后应能正常显示。

## 2. 首页底部提示移除

不再显示：

```text
保存済みの写真を選択してください。
```

首页只保留主要操作说明和选择照片按钮。

## 3. 权限表述三语言统一

权限相关表述统一为“允许所有权限”，不再强调传感器。

```text
日本語：すべての権限を許可してください。
中文：请允许所有权限。
English：Allow all permissions.
```

兼容性说明中也统一为“如出现权限请求，请允许所有权限”。

## 4. 版本号更新

资源路径更新为：

```text
app.js?v=2.11.0
styles.css?v=2.11.0
manifest.webmanifest?v=2.11.0
```

## 5. 保持不变

- v1.6 的矩阵相对旋转传感器逻辑保持不变
- 独立 1080p 录制画布保持
- WebM 时长元数据修正保持
- 日语默认和中日英切换保持
- 拍照功能继续移除，只保留照片导入
