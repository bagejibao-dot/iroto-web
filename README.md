# Iroto Web v2.6

这是 Iroto Web v2.6：语言按钮形状与日语文案排版修正版。

## 1. 语言选择恢复成长孔型按钮

v2.5 为了去掉下拉箭头，把语言按钮做得过窄，English 会显得太满。  
v2.6 改为：

```text
长孔型 pill 按钮
无下拉小箭头
文字居中
宽度加大到约 110–118px
```

## 2. 首页日语语气调整

首页文案从：

```text
傾けて演奏します。
```

改为：

```text
傾けて演奏しましょう。
```

语气更适合体验说明。

## 3. 日语弹窗文案缩短

为了减少竖屏时末尾一个字单独换行的问题，缩短了部分日语：

```text
新しい演奏用の写真を選択してください。
→ 演奏用の写真を選択してください。

録画データを削除しますか？この操作は元に戻せません。
→ 録画を削除しますか？
```

兼容性说明也略微缩短，避免长句在窄屏幕上出现不自然换行。

## 4. 版本号更新

资源路径更新为：

```text
app.js?v=2.6.0
styles.css?v=2.6.0
manifest.webmanifest?v=2.6.0
```

方便浏览器自动获取新版。

## 5. 保持不变

- v1.6 的矩阵相对旋转传感器逻辑保持不变
- 独立 1080p 录制画布保持
- WebM 时长元数据修正保持
- 日语默认和中日英切换保持
