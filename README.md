# Iroto Web v2.3

这是 Iroto Web v2.3：演奏显示比例回退 + 独立 1080p 录制画布版。

## 1. 演奏界面恢复照片原比例

v2.2 把演奏界面和录制画面一起改成了 cover，导致一进入演奏界面照片被铺满裁切。  
v2.3 已改回：

```text
演奏界面：contain
照片保持原比例
不拉伸
不裁切
```

也就是画面显示回到之前“照片归照片”的方式。

## 2. 录制改用独立画布

录制不再直接捕获屏幕上的 canvas。  
v2.3 新增一个专门用于录制的隐藏 canvas：

```text
演奏显示 canvas：按手机屏幕显示，照片保持原比例
录制 canvas：按照片原始画幅生成，接近 1080p 画质
```

这样可以避免“手机屏幕画幅”和“照片画幅”不同导致录制视频出现黑边。

## 3. 录制画幅规则

录制画布会根据照片比例自动生成接近 1080p 的尺寸：

```text
16:9 横图：约 1920 × 1080
9:16 竖图：约 1080 × 1920
4:3 横图：约 1440 × 1080
3:4 竖图：约 1080 × 1440
1:1 方图：约 1080 × 1080
```

如果照片特别宽或特别高，会把长边限制在约 1920，避免文件过大。

## 4. 录制内容

录制视频中会包含：

```text
照片
准星
轨迹
音名 / Rest
当前演奏状态
```

但不会录入网页顶部/底部 UI、语言按钮、兼容性按钮或录制计时器。

## 5. 保持不变

- v1.6 的矩阵相对旋转传感器逻辑保持不变
- 录制计时器保持
- 日语默认和中日英切换保持
- 保存信息多语言和删除二次确认保持
- Service Worker 仍然禁用，便于 GitHub Pages / Netlify 测试
