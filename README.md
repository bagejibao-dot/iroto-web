# Iroto Web v2.5

这是 Iroto Web v2.5：缓存更新友好化 + 首页排版与语言按钮细节修正版。

## 1. 减少更新时必须清缓存的问题

为了避免以后给别人测试时，对方还需要手动清除缓存，v2.5 给主要资源加了版本号：

```text
styles.css?v=2.5.0
app.js?v=2.5.0
manifest.webmanifest?v=2.5.0
```

以后发布新版本时，只要版本号变更，浏览器会把 JS / CSS 当作新资源重新请求。  
这比让测试者手动清除缓存方便很多。

同时，index.html 中也加入了 no-cache 的 meta hints。  
注意：GitHub Pages 不能像 Netlify `_headers` 那样完全自定义 HTTP cache header，所以极端情况下浏览器可能短时间保留旧 index.html；但资源版本号已经能解决大部分“JS/CSS 不更新”的问题。

## 2. 语言切换按钮去掉下拉箭头

语言选择仍然会弹出系统选择菜单，但按钮本体不再显示小箭头。  
文字改为居中显示，更像普通圆角按钮。

## 3. 首页文字重新排版

首页主说明缩短，并按语义换行：

```text
写真を選択または撮影し、
傾けて演奏します。
```

浏览器推荐说明移动到首页底部。

## 4. 传感器权限文案调整

首页提示改为“允许所有传感器权限”：

```text
すべてのセンサー権限を許可してください。
请允许所有传感器权限。
Allow all sensor permissions.
```

## 5. 保持不变

- v1.6 的矩阵相对旋转传感器逻辑保持不变
- 演奏界面照片保持原比例显示
- 录制使用独立照片比例的接近 1080p canvas
- WebM 时长元数据修正保持
- 日语默认和中日英切换保持
