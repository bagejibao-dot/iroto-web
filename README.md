# Iroto Web v2.15

这是 v2.15 版本。  
基底来自 v2.14.8，但修复了主页横屏图标和兼容性检查弹窗问题。

## 1. 主页图标修复

v2.14.6 的主页图标由多个嵌套 `span` 叠加构成，横屏压缩时容易出现嵌套和缩放错位。  
v2.15 将图标改为单个内联 SVG，视觉上保持 v2.14.6 的设计方向，但结构更稳定。

## 2. 兼容性检查弹窗修复

v2.15 重新整理兼容性检查弹窗：

```text
版本号放到标题右侧，确保显示
每个检查项使用纵向布局，避免右侧状态列挤压文字
说明文字自然换行，不再被边距裁切
小屏幕下缩小内边距和字号
```

底部不再显示额外测试标记，只显示普通版本号：

```text
v2.15
```

## 3. 保持不变

没有修改以下核心逻辑：

```text
全屏逻辑
方向锁定
传感器控制
准星平滑
录制逻辑
照片导入逻辑
点击画面显示 / 隐藏 UI
```
