# Iroto Web v2.12

这是 Iroto Web v2.12：取消自动全屏版。

## 1. 开始演奏时不再主动进入浏览器全屏

移动浏览器进入 Fullscreen API 全屏时，会显示系统/浏览器自己的提示，例如“如何退出全屏”。  
这个提示属于浏览器安全提示，网页代码不能直接隐藏。

因此 v2.12 改为：

```text
播放时不再自动 requestFullscreen()
保持当前网页显示状态
继续保留演奏 UI、传感器、录制等功能
```

这样开始演奏时不会再触发底部全屏退出提示。

## 2. 说明

如果用户自己通过浏览器菜单或系统方式进入全屏，那仍可能出现浏览器提示；但 Iroto Web 本身不再主动触发全屏。

## 3. 版本号更新

资源路径更新为：

```text
app.js?v=2.12.0
styles.css?v=2.12.0
manifest.webmanifest?v=2.12.0
```

## 4. 保持不变

- v1.6 的矩阵相对旋转传感器逻辑保持不变
- 照片导入与显示修复保持
- 首页底部提示继续隐藏
- 权限文案继续统一为“允许所有权限”
- 独立 1080p 录制画布保持
- WebM 时长元数据修正保持
