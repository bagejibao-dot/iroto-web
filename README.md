# Iroto Web v2.7

这是 Iroto Web v2.7：语言按钮尺寸、手机拍照方式与横屏首页布局修正版。

## 1. 语言选择按钮缩回以前的长孔大小

v2.6 的语言按钮过长。  
v2.7 改回接近以前带下拉箭头时的长孔尺寸：

```text
桌面 / 普通屏：约 96px
小屏手机：约 92px
无下拉箭头
文字居中
```

## 2. 手机拍照不再打开网页内相机弹窗

之前横屏时网页相机弹窗可能显示不全，导致无法点击拍照。  
v2.7 在手机上改为直接调用系统相机 / 文件选择：

```text
手机：使用系统相机或系统文件选择界面
桌面：仍可使用网页内相机弹窗
```

这样横屏手机上不会再被网页弹窗尺寸限制。

## 3. 横屏首页紧凑布局

横屏手机高度较低时，首页文字和按钮容易重叠。  
v2.7 加入横屏紧凑布局：

```text
logo 缩小
标题缩小
说明文字缩短行高
按钮高度压缩
底部浏览器提示放低并缩小
整体使用 safe-area 避开刘海 / 前置摄像头区域
```

## 4. 版本号更新

资源路径更新为：

```text
app.js?v=2.7.0
styles.css?v=2.7.0
manifest.webmanifest?v=2.7.0
```

## 5. 保持不变

- v1.6 的矩阵相对旋转传感器逻辑保持不变
- 独立 1080p 录制画布保持
- WebM 时长元数据修正保持
- 日语默认和中日英切换保持
